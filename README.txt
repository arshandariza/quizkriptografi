Program Vigenere Cipher dengan GUI Berbasis Tkinter
Deskripsi
Program ini merupakan implementasi Vigenere Cipher untuk enkripsi dan dekripsi teks, yang dilengkapi dengan antarmuka grafis (GUI) menggunakan Tkinter. Program ini dapat menerima input teks langsung melalui keyboard atau melalui upload file .txt. Kunci yang digunakan harus minimal sepanjang 12 karakter.

Fitur:
Enkripsi teks menggunakan Vigenere Cipher.
Dekripsi teks yang telah dienkripsi.
Input teks manual atau melalui file .txt.
Hasil enkripsi atau dekripsi ditampilkan di area output.
Persyaratan Sistem
Python 3.x harus terinstall di sistem Anda.
Modul Tkinter (biasanya sudah termasuk dalam instalasi Python).
Cara Menggunakan Program
Langkah 1: Instalasi Python (jika belum terinstall)
Pastikan Python 3 sudah terinstall di komputer Anda. Anda dapat memeriksanya dengan menjalankan perintah berikut di terminal/command prompt:

css
Salin kode
python --version
Jika belum terinstall, silakan download dan install Python dari situs resmi Python.

Langkah 2: Menjalankan Program
Download atau Clone Kode Program:

Download file Python dari repository atau clone repository dari GitHub dengan perintah:
bash
Salin kode
git clone https://github.com/username/repository-name.git
Buka Terminal/Command Prompt:

Navigasi ke direktori tempat program berada.
Jalankan Program:

Jalankan file Python dengan perintah:
Salin kode
python nama_file_program.py
(Sesuaikan nama_file_program.py dengan nama file Python Anda)

Langkah 3: Menggunakan Antarmuka Program
Setelah program dijalankan, antarmuka grafis akan terbuka. Berikut adalah cara menggunakannya:

Input Teks Manual:

Masukkan teks yang ingin dienkripsi atau didekripsi di area teks yang tersedia.
Upload File Teks (.txt):

Jika ingin menggunakan file teks, klik tombol Upload File dan pilih file .txt yang ingin dienkripsi atau didekripsi.
Masukkan Kunci:

Masukkan kunci untuk enkripsi atau dekripsi di kolom yang tersedia. Kunci harus minimal 12 karakter.
Proses Enkripsi/Dekripsi:

Klik tombol Enkripsi untuk mengenkripsi teks, atau klik tombol Dekripsi untuk mendekripsi teks.
Hasil enkripsi atau dekripsi akan ditampilkan di area output di bawahnya.
Catatan Penting:
Kunci harus memiliki panjang minimal 12 karakter.
File teks yang diunggah harus dalam format .txt.
